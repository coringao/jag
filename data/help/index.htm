How to play

Move game pieces into matches of 3 or more in a straight line horizontally or vertically. Doing this on top of "target" cells will break them.

The aim of JAG is to break all of the "targets" on each level, and to do this before the time runs out. Keep doing this until you have beaten the last level and won the game.

Scoring

You get score points for removing game pieces. Basically, you get 1 point for the ordinary cell and 3 points for the "target" cell. The actual points you get are multiplyed by bonus value, so it can be much higher.

Bonus system

Bonus is the multiplier which is applyed to every score point you get during the game. The bonus raises when you remove "target" cells quickly enough to keep bonus timer run. By default bonus timer set to 5 seconds, during the game you can upgrade this value.

To permanently upgrade multiplier value, use the Score-Magnification Bonus.

To permanently upgrade timer value, use the Score-Time Bonus.

Items

Every level may contain up to 6 game items. The icon of an item depends on current graphic theme selected.

In arcade mode there is a reward, which is given for 500 or more pieces of collected items - you can eliminate all the items of the collected type from the field. You can choose the power of the elimination depending of how many items you collected.

Tools

Tools are the special rewards that you take for scoring and collecting game items. There are 10 tools available in the game (2 in the puzzle mode):
• Hammer - destroys an item in the game field
• Nippers - removes a block
• Star - removes item, block and one target
• Clock - adds +60 seconds to the level timer
• Super star - removes everything from a cell
• Bomb - explodes a region of 9 cells
• Mixer - randomly mixes items in the field
• Row Blaster - removes items, blocks and targets in a row
• Bolt - randomly removes items from the field
• Twin Blaster - removes all items of the selected type

Permanent Bonus

Unlike tools, permanent bonus, once received, is active for all the further levels. Getting permanent bonus of the same type several times increases its power.
• Score-Magnification Bonus - permanent +1 to multiplier value
• Score-Time Bonus - permanent extra 1 seconds to bonus timer
• Score Bonus - permanent extra 1 point for the target cell
• Clock Bonus - permanent extra 30 seconds to level timer

Arcade vs Puzzle

There are two play modes in the game - arcade and puzzle. The difference is - in arcade mode you can get bonuses for collected items and use all the tools, and in the puzzle mode you have only two tools. Also in the puzzle mode level doesn't fail when the time is over, you may continue, but won't get any bonus points if the time is zero.
